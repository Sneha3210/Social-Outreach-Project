About This Project

The Personal Health Status Analyzer is designed as a social impact project to help people identify preventive health risks using simple parameters like BMI, lifestyle, and sleep.

This tool provides personalized nutrition and fitness recommendations, aiming to improve wellness accessibility for underserved populations.
